## Migration to Homebrew Core

Follow these steps to switch to the version maintained in Homebrew core:

```shell
# 1. Uninstall the tap version and remove the tap
brew uninstall zsv
brew untap liquidaty/zsv

# 2. Update Homebrew and install from core
brew update
brew install zsv
```
